# 0x0F. C - Function pointers
## Description
This sub-module of ALX low level programming introduces the concept of Function pointers in C programming language. It gives an understanding of the following:-
- What are function pointers and how to use them
- What does a function pointer exactly hold
- Where does a function pointer point to in the virtual memory

## Requirement
### General
- All your files will be compiled on Ubuntu 20.04 LTS using gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
- All your files should end with a new line
- Your code should use the Betty style. It will be checked using betty-style.pl and betty-doc.pl
- You are not allowed to use global variables
- No more than 5 functions per file
- The only C standard library functions allowed are malloc, free and exit. Any use of functions like printf, puts, calloc, realloc etcâ€¦ is forbidden
- You are allowed to use _putchar
- The prototypes of all your functions and the prototype of the function _putchar should be included in your header file called function_pointers.h
- All your header files should be include guarded
